Projeto 03

Dashboards mentem. Dados contam histórias.

BI como narrativa crítica, não como decoração.

Contexto

Painéis neutros escondiam tensões reais. A leitura crítica precisava de narrativa, não apenas de indicadores isolados.

Desafios

Construir hierarquia visual sem simplificar demais, garantir leituras compartilhadas e devolver contexto social aos números.

Resultados

Dashboards com camadas narrativas, perguntas-guia e decisões mais conscientes sobre impacto e responsabilidade.
